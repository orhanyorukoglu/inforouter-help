## Key Features

infoRouter provides a comprehensive set of features for document management:

- **Fast Document Search** - Amazingly fast and accurate document searching
- **Version Control** - Track document versions and changes
- **Security** - Flexible, integrated security for documents and folders
- **Workflow Management** - Document-based workflow management
- **Records Management** - DoD 5015.2 STD compliant records management
- **Portal Interface** - Multiple portal interfaces for different user types
- **Web Services API** - Develop custom interfaces to in-house systems
- **Email Integration** - MS Outlook integration and email management
- **Office Integration** - Seamless MS Office integration
