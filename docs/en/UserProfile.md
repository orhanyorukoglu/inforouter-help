# User Profile

The User Profile screen contains information about your infoRouter account. Access it from your **Personal Dashboard** or, if you're an administrator, through **User Management** in the Control Panel.

![User Profile screen](../images/User_Profile_1.jpg)
![My Profile screen](../images/my_profile_screen.jpg)

---

## Profile Sections

The User Profile screen has four main sections:

1. **General Profile** — Basic account information
2. **Settings and Preferences** — Personalization options
3. **Task Redirection** — Delegate tasks during absence
4. **Profile Sidebar** — Quick links to related information

---

## General Profile

This section displays your basic account information.

!!! note "Limited Editing"
    As a user, you can only change your **password** in this section. All other parameters are managed by the System Administrator or User Managers.

### Changing Your Password

1. Click the **properties icon** next to the password field
2. Enter your new password
3. Confirm the new password
4. Click **Save**

!!! tip "Password Security"
    - Choose a password that is difficult to guess but easy to remember
    - Usernames cannot be changed
    - Even administrators cannot see your password
    - If you forget your password, use the "Forgot Password" link or contact your System Administrator

## Settings and Preferences

You have full control over these settings:

| Setting | Description |
|---------|-------------|
| **Language Preference** | Sets the language for all infoRouter messages and labels |
| **Default Portal** | Portal you're redirected to after logging in |
| **E-mail** | Email address for notifications |
| **Notification Type** | How you receive notifications |
| **E-mail Type** | Format of email notifications |
| **Attach Documents** | Include documents in email notifications |

### Notification Options

| Type | Description |
|------|-------------|
| **None** | Disable all email notifications |
| **Instant** | Receive notifications immediately as events occur |
| **Daily** | Receive a consolidated daily report of all notifications |

### Email Format Options

| Format | Description |
|--------|-------------|
| **HTML** | Rich formatted emails with styling |
| **TEXT** | Plain text emails |

---

## Task Redirection

Redirect task assignments to another user during your absence (vacation, leave, etc.).

!!! warning "Important"
    Task redirection only applies to **future** task assignments. Existing tasks are not affected.

## Profile Sidebar

The sidebar provides quick access to related information:

| Link | Description | Visibility |
|------|-------------|------------|
| :material-account-group: **Global Group Memberships** | Global user groups you belong to | Admin only |
| :material-library: **Library Memberships** | Libraries you have access to | Admin only |
| :material-account-multiple: **Local Group Memberships** | Local groups by library | All users |
| :material-bell: **Subscriptions** | Documents you're subscribed to | All users |
| :material-file-document-multiple: **Documents and Folders** | Items you own | All users |
| :material-file-edit: **Checked out documents** | Documents currently checked out to you | All users |
| :material-pencil: **Authored documents** | Documents you have authored | All users |
| :material-eye: **View Log** | Documents you have viewed | All users |
| :material-sitemap: **Workflow Roles** | Folders where you're assigned as reviewer | All users |
| :material-email-fast: **Send welcome email** | Send welcome email to new users | Admin only |
