# Renaming Folders

Change folder names while preserving all content and settings.

---

## How to Rename a Folder

### Method 1: Folder Properties

1. Navigate to the folder
2. Right-click or click the **Action** icon
3. Select **Folder Properties**
4. Edit the **Folder Name** field
5. Click **OK**

### Method 2: Edit Menu

1. Select the folder by clicking the checkbox
2. Go to **Edit** â†’ **Rename**
3. Enter the new name
4. Press **Enter** or click **OK**

| Change permission | :material-check: Yes |
| Read only | :material-close: No |

## Naming Guidelines

!!! warning "Folder Names"
    Avoid these characters in folder names:

    `\ / : * ? " < > |`
