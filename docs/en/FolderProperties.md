# Folder Properties

View and modify folder metadata, settings, and configuration options.

---

## Accessing Folder Properties

1. Navigate to the folder
2. Right-click on the folder, or click the **Action** icon
3. Select **Folder Properties**

---

## Basic Information

| Property | Description |
|----------|-------------|
| **Folder Name** | Display name of the folder |
| **Description** | Optional description of folder contents |
| **Owner** | User who owns the folder |
| **Created** | Date folder was created |
| **Created By** | User who created the folder |

---

## Location Information

| Property | Description |
|----------|-------------|
| **Library** | Parent library containing the folder |
| **Path** | Full path to the folder |
| **Parent Folder** | Immediate parent folder |

---

## Statistics

| Property | Description |
|----------|-------------|
| **Documents** | Number of documents in folder |
| **Subfolders** | Number of subfolders |
| **Total Size** | Combined size of all contents |

---

## Sidebar Options

The folder properties sidebar provides access to:

| Option | Description |
|--------|-------------|
| :material-shield-lock: **Security** | Manage folder permissions |
| :material-file-multiple: **Custom Properties** | Add/edit metadata |
| :material-bell: **Subscriptions** | Subscribe to folder changes |
| :material-calendar-lock: **Retention** | Set retention schedule |
| :material-clipboard-list: **Audit Log** | View folder activity |

---

## Editing Properties

1. Open Folder Properties
2. Modify the desired fields
3. Click **OK** to save changes

!!! info "Permission Required"
    You need **Change** or **Full Control** permission to modify folder properties.
