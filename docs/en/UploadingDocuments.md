# Uploading Documents

Add documents to infoRouter by uploading files from your computer.

---

## How to Upload Documents

### Method 1: New Menu

1. Navigate to the destination folder
2. Click **New** in the menu bar
3. Select **Upload**
4. Click **Browse** to select files
5. Optionally set document properties
6. Click **Upload**

### Method 2: Drag and Drop

1. Navigate to the destination folder
2. Drag files from your computer
3. Drop them into the documents panel

---

## Upload Options

| Option | Description |
|--------|-------------|
| **Document Name** | Display name (defaults to filename) |
| **Description** | Optional document description |
| **Document Type** | Category (Invoice, Report, etc.) |
| **Keywords** | Search keywords |
| **Custom Properties** | Metadata fields |

---

## Supported File Types

infoRouter supports virtually all file types:

| Category | Examples |
|----------|----------|
| :material-microsoft-office: **Office** | .docx, .xlsx, .pptx |
| :material-file-pdf-box: **PDF** | .pdf |
| :material-image: **Images** | .jpg, .png, .gif |
| :material-text: **Text** | .txt, .rtf |
| :material-archive: **Archives** | .zip, .rar |

---

## File Size Limits

!!! info "Size Restrictions"
    Maximum file size is determined by system settings. Contact your administrator if you encounter size limits.

---

## Permissions Required

| Permission | Can Upload |
|------------|------------|
| Add | :material-check: Yes |
| Add & Read | :material-check: Yes |
| Change | :material-check: Yes |
| Full Control | :material-check: Yes |
| Read only | :material-close: No |

---

## After Upload

Uploaded documents:

- Appear in the destination folder
- Can be viewed immediately
- Inherit folder security settings
- Are indexed for search
