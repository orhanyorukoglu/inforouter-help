# Search Introduction

Find documents quickly using infoRouter's powerful search capabilities.

---

## Search Options

infoRouter offers multiple ways to find documents:

| Search Type | Best For |
|-------------|----------|
| :material-magnify: **[Basic Search](BasicSearch.md)** | Quick keyword searches |
| :material-magnify-plus-outline: **[Advanced Search](AdvancedSearch.md)** | Complex criteria and filters |
| :material-folder-search: **[Categories](Categories.md)** | Pre-defined search queries |
| :material-content-save-search: **[Saved Searches](CreatingSavedSearches.md)** | Frequently used searches |

---

## What Can You Search?

| Content Type | Searchable |
|--------------|------------|
| :material-file-document: **Document content** | :material-check: Yes (full-text) |
| :material-tag: **Document names** | :material-check: Yes |
| :material-card-text: **Properties/metadata** | :material-check: Yes |
| :material-comment: **Comments** | :material-check: Yes |
| :material-form-textbox: **Custom properties** | :material-check: Yes |

---

## Search Scope

Control where infoRouter searches:

| Scope | Description |
|-------|-------------|
| **Current folder** | Only the selected folder |
| **Current library** | Entire library |
| **All libraries** | All libraries you have access to |

---

## Tips for Better Results

!!! tip "Search Effectively"
    - Use specific keywords
    - Use quotes for exact phrases: `"annual report"`
    - Combine with property filters for precision
    - Use [search operators](SearchOperators.md) for complex queries
