# Archiving Document Libraries

Move entire document libraries offline while keeping them accessible when needed. Archived libraries are hidden from the normal interface but remain fully accessible.

---

## Who Can Archive Libraries?

| Role | Can Archive |
|------|-------------|
| System Administrator | :material-check: Yes |
| Administrators group | :material-check: Yes |
| Library Managers | :material-close: No |
| Regular Users | :material-close: No |

---

## How to Archive a Library

1. Click the **Control Panel** tab
2. Click **Libraries**
3. Select the library you want to archive
4. Click **Archive this Library**
5. Confirm the archiving operation

!!! warning "Check-In Required"
    All checked out documents must be checked in before a library can be archived. Ensure no documents are currently checked out.

---

## Viewing Archived Libraries

Archived libraries are hidden by default. To view them:

1. Go to your user settings
2. Enable **Show Archives**
3. Archived libraries appear in **purple** to distinguish them from online libraries

---

## Unarchiving a Library

To bring a library back online:

1. Click the **Control Panel** tab
2. Click **Libraries**
3. Enable **Show Archives** if needed
4. Select the archived library
5. Click **Unarchive this Library**

---

## Archive Considerations

| Consideration | Details |
|---------------|---------|
| :material-magnify: **Search** | Archived libraries are excluded from default searches |
| :material-eye: **Visibility** | Hidden from normal view until Show Archives is enabled |
| :material-lock: **Security** | All security settings are preserved |
| :material-file-document: **Content** | All documents and folders remain intact |
