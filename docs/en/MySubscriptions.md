# My Subscriptions

View and manage all your document and folder subscriptions in one place.

---

## What Are Subscriptions?

Subscriptions let you receive automatic notifications when changes occur to documents or folders you care about.

---

## Accessing My Subscriptions

Click **My Subscriptions** in the Personal Dashboard to view:

- All documents you're subscribed to
- All folders you're subscribed to
- Subscription settings for each item

---

## Subscription List View

| Column | Description |
|--------|-------------|
| :material-file-document: **Name** | Document or folder name |
| :material-folder: **Location** | Where the item is stored |
| :material-calendar: **Subscribed** | When you subscribed |
| :material-bell: **Notifications** | What events you're tracking |

---

## Document Subscriptions

When you subscribe to a document, you're notified of:

| Event | Notification |
|-------|--------------|
| :material-pencil: **Document modified** | Content changes |
| :material-file-multiple: **New version** | Version added |
| :material-comment-plus: **New comment** | Comment added |
| :material-tag-edit: **Properties changed** | Metadata updated |

---

## Folder Subscriptions

When you subscribe to a folder, you're notified of:

| Event | Notification |
|-------|--------------|
| :material-file-plus: **New document** | Document added to folder |
| :material-pencil: **Document modified** | Existing document changed |
| :material-folder-plus: **New subfolder** | Subfolder created |

---

## Managing Subscriptions

From My Subscriptions, you can:

| Action | How |
|--------|-----|
| :material-eye: **View details** | Click on item name |
| :material-cog: **Change settings** | Click Settings icon |
| :material-bell-off: **Unsubscribe** | Click Unsubscribe |
