# Categories / Editing Categories

Only designated "Search and Category Administrators" can perform the following actions.

To edit Categories, simply perform the following:

1. Navigate to the "Categories" tab.
2. Click on the small "Edit" icon located next to the Category name in the Categories page.
3. The search window used to create the category will appear.
4. Make the desired changes to the category and click on the "Save" button.
