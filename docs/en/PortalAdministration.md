# Portal Administration

Configure and manage infoRouter portals to create customized user experiences.

---

## Who Can Administer Portals?

| Role | Access |
|------|--------|
| System Administrator | :material-check: Full access |
| Portal Administrators | :material-check: Assigned portals |
| Administrators group | :material-check: Full access |
| Regular Users | :material-close: No access |

---

## Portal Configuration Options

### Branding

| Setting | Description |
|---------|-------------|
| **Logo** | Custom portal logo |
| **Colors** | Theme color scheme |
| **Header/Footer** | Custom content areas |
| **CSS** | Advanced styling |

### Content

| Setting | Description |
|---------|-------------|
| **Libraries** | Which libraries are visible |
| **Categories** | Available search categories |
| **Quick Links** | Shortcut links |
| **Announcements** | News and updates |

### Access

| Setting | Description |
|---------|-------------|
| **Users** | Who can access the portal |
| **Default portal** | Set as user's home |
| **Anonymous access** | Public access settings |

---

## Creating a New Portal

1. Navigate to **Control Panel**
2. Click **Portal Administration**
3. Click **New Portal**
4. Configure portal settings
5. Assign users or groups
6. Save changes

---

## Managing Existing Portals

| Action | Description |
|--------|-------------|
| :material-pencil: **Edit** | Modify portal settings |
| :material-account-plus: **Add Users** | Assign users to portal |
| :material-eye: **Preview** | View portal as user sees it |
| :material-delete: **Delete** | Remove portal |
