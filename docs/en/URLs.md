# URL Documents

URL documents are bookmarks that link to external or internal web resources. They are treated as documents within infoRouter and can be organized, secured, and searched like any other document.

---

## What Are URL Documents?

URL documents store links to:

- External websites
- Internal web applications
- SharePoint sites
- Other web resources

!!! tip "Centralized Links"
    Use URL documents to create a centralized, organized collection of important web links for your team.

---

## Creating a URL Document

1. Navigate to the destination folder
2. Click **New** > **URL**
3. Enter the following:

| Field | Description |
|-------|-------------|
| **Name** | Display name for the URL |
| **URL** | The web address (http:// or https://) |
| **Description** | Optional description |
| **Open in new window** | Whether to open in a new browser tab |

4. Click **OK**

---

## URL Document Behavior

| Action | Result |
|--------|--------|
| :material-cursor-default-click: **Click URL** | Opens the linked web page |
| :material-shield: **Security** | Same as regular documents |
| :material-magnify: **Search** | Name and description are searchable |
| :material-folder-move: **Move/Copy** | Works like regular documents |

---

## Use Cases

| Scenario | Example |
|----------|---------|
| **Reference Links** | Links to vendor documentation |
| **Internal Tools** | Links to company applications |
| **Training Resources** | Links to online training materials |
| **Client Portals** | Links to client-specific sites |

---

## URL vs Shortcuts

| Feature | URL Document | Shortcut |
|---------|--------------|----------|
| **Points to** | External web address | Internal infoRouter document |
| **Opens** | Web browser | Document in native app |
| **Security** | Own permissions | Inherits from target |
