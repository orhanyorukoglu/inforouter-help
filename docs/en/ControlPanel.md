# Control Panel

The Control Panel provides access to all administrative functions in infoRouter. Only users with administrative privileges can access this area.

---

## Accessing the Control Panel

Click the **Control Panel** tab in the main navigation bar.

!!! note "Access Requirements"
    The Control Panel tab only appears if you have administrative privileges (System Administrator, Library Manager, or member of an administrative group).
