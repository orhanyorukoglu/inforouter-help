# Import/Export Tool

The infoRouter Import/Export Tool is a desktop application for bulk document transfers between your file system and infoRouter.

---

## Overview

This tool provides a graphical interface for:

- Bulk importing documents into infoRouter
- Exporting documents from infoRouter
- Preserving folder structures during transfer
- Applying metadata during import

---

## Key Features

| Feature | Description |
|---------|-------------|
| :material-upload-multiple: **Bulk Import** | Upload many documents at once |
| :material-download-multiple: **Bulk Export** | Download documents from infoRouter |
| :material-folder-multiple: **Folder Structure** | Preserve hierarchies during transfer |
| :material-form-textbox: **Metadata** | Apply properties during import |
| :material-api: **Web Services** | Uses API for reliable transfers |

---

## Common Use Cases

| Scenario | Description |
|----------|-------------|
| :material-database-import: **Initial Migration** | Move existing documents to infoRouter |
| :material-backup-restore: **Backup** | Export documents for offline storage |
| :material-folder-upload: **Bulk Upload** | Add large document sets |
| :material-sync: **Synchronization** | Keep local and infoRouter copies in sync |

---

## Getting the Tool

1. Log into infoRouter
2. Go to **Tools** menu
3. Select **Client Add-ins and Programs**
4. Download **Import/Export Tool**
