# Filter Menu

Filter the document list to find specific items within a folder.

---

## Filter Options

| Filter | Description |
|--------|-------------|
| :material-text-search: **Containing Text** | Search documents by content |
| :material-form-textbox: **Named** | Search by document or folder name |
| :material-filter-variant: **Search for** | Filter by type (Folders, Documents, or both) |
| :material-file-document: **Document Type** | Filter by assigned document type |
| :material-file-question: **Format** | Filter by file format (images, videos, Office, etc.) |

---

## How to Use Filters

1. Click the **Filter** menu
2. Enter values in one or more filter fields
3. Click the **Filter** button
4. Results display in the document list

!!! tip "Combining Filters"
    Use multiple filter options together to narrow your search results.

---

## Example Use Cases

| Goal | Filters to Use |
|------|----------------|
| Find all PDFs | Format: PDF |
| Find invoices containing "2024" | Document Type: Invoice + Containing Text: 2024 |
| Find folders only | Search for: Folders |
| Find images by name | Format: Images + Named: (search term) |
