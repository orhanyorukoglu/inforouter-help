# Custom Property Sets / Applying Custom Property Sets

Applying Custom Properties to Folders:

1. Click on the "Action" icon and choose the "Folder Properties" menu
2. The "Folder Properties" window will appear.
3. Scroll down to the section labeled "Custom Properties"
4. Select a "Custom Property" from the drop down list.
5. The "Custom Property Entry" window will appear.
6. Enter values for each prompt in the "Custom Property"
     
   **Note:** Some values may be required. Required values will be clearly marked.
7. Check the box labeled "Apply to sub-folder and documents" link if you wish to apply the entered values to the documents within the folder and subfolders.
8. Click "Ok" to apply the values.

Applying Custom Properties to Documents:

1. Click on the "Action" icon and choose the "Document Properties" menu.
2. The "Document Properties" window will appear.
3. Scroll down to the section labeled "Custom Properties"
4. Select a "Custom Property" from the drop down list.
5. The "Custom Property Entry" window will appear.
6. Enter values for each prompt in the "Custom Property"
     
   **Note:** Some values may be required. Required values will be clearly marked.
7. Click "Ok" to apply the values.

Security Considerations:

By default, you must have at least "Change" permissions to apply custom Document property values. The library policies may indicate higher requirements or completely disallow the application of custom property values by regular users.
