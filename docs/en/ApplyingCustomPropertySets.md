# Applying Custom Property Sets

Add metadata to folders and documents using custom property sets.

---

## Applying to Folders

1. Click the **Action** icon on the folder
2. Select **Folder Properties**
3. Scroll to the **Custom Properties** section
4. Select a custom property set from the dropdown
5. Enter values for each field
6. Optionally check **Apply to sub-folders and documents**
7. Click **OK**

!!! tip "Bulk Application"
    Use the "Apply to sub-folders and documents" option to apply the same property values to all contents within the folder.

---

## Applying to Documents

1. Click the **Action** icon on the document
2. Select **Document Properties**
3. Scroll to the **Custom Properties** section
4. Select a custom property set from the dropdown
5. Enter values for each field
6. Click **OK**

---

## Required Fields

!!! warning "Required Values"
    Some custom property fields may be required. These will be clearly marked and must be completed before saving.

---

## Permissions Required

| Permission | Can Apply Custom Properties |
|------------|----------------------------|
| :material-pencil: Change | :material-check: Yes (default minimum) |
| :material-shield: Full Control | :material-check: Yes |
| :material-eye: Read only | :material-close: No |

!!! note "Library Policies"
    Library policies may require higher permissions or restrict custom property application entirely.
