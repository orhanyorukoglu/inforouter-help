# The Menu System

The infoRouter menu system provides access to all document and folder operations. This section describes each menu and its options in detail.

---

## Main Menus

| Menu | Description |
|------|-------------|
| :material-plus: **[New Menu](MenuNew.md)** | Create new documents, folders, and other items |
| :material-pencil: **[Edit Menu](MenuEdit.md)** | Check in/out, copy, move, delete operations |
| :material-wrench: **[Tools Menu](MenuTools.md)** | Utilities and advanced features |
| :material-eye: **[View Menu](MenuView.md)** | Display options and column selection |
| :material-menu: **[Custom Menus](MenuCustom.md)** | Organization-specific menu options |

---

## Context Menus

In addition to the main menu bar, infoRouter provides context-sensitive menus:

- **Action Menu** — Click the Action icon on any document or folder row
- **Right-click Menu** — Right-click on items for quick access to common actions

---

## Menu Availability

!!! note "Permission-Based Menus"
    Menu options are displayed based on your permissions. You will only see options that you are authorized to use for the selected item.
