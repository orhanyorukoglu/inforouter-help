# Folder Retention

Protect folders from deletion by applying retention schedules that enforce compliance requirements.

---

## What Is Folder Retention?

Folder retention applies a Retention and Disposition schedule to a folder, preventing its deletion until the defined retention period has elapsed.

---

## Benefits of Folder Retention

| Benefit | Description |
|---------|-------------|
| :material-shield: **Deletion protection** | Folder cannot be deleted during retention |
| :material-file-multiple: **Cascading protection** | Applies to all documents within |
| :material-gavel: **Compliance** | Meets legal and regulatory requirements |
| :material-calendar-check: **Automated tracking** | System tracks retention dates |

---

---

## Folder vs Document Retention

| Scope | Description |
|-------|-------------|
| **Folder Retention** | Protects the folder itself from deletion |
| **Document Retention** | Protects individual documents |

!!! info "Combined Protection"
    You can apply retention schedules at both folder and document levels for comprehensive protection.

---

## Retention Schedule Options

| Option | Description |
|--------|-------------|
| **Retention Period** | How long folder must be retained |
| **Disposition Action** | What happens when retention ends |
| **Assignee** | User responsible for disposition |
