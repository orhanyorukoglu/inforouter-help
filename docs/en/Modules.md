# Modules and Add-ins

Extend infoRouter's capabilities with optional modules for document capture and processing.

---

## Available Modules

| Module | Description |
|--------|-------------|
| :material-microsoft-office: **[Office Add-in](OfficeAddin.md)** | Save documents directly from Microsoft Office applications |
| :material-folder-arrow-down: **[Hot Folders](HotFolders.md)** | Automatically import files from watched folders |
| :material-scanner: **[Scan Station](ScanStation.md)** | Scan paper documents directly into infoRouter |
| :material-swap-horizontal: **[Import/Export Tool](ImportExport.md)** | Bulk import/export documents and folders |
| :material-email-sync: **[E-mail Scanner](EmailScanner.md)** | Capture emails from POP3 accounts automatically |

---

## Module Capabilities

### Office Add-in

| Feature | Benefit |
|---------|---------|
| Save from Word/Excel/PowerPoint | Direct document upload |
| Version control integration | Seamless check-in/out |
| Metadata entry | Add properties at save time |

### Hot Folders

| Feature | Benefit |
|---------|---------|
| Automatic monitoring | No manual uploads needed |
| Scheduled processing | Configure timing |
| Metadata templates | Auto-assign properties |

### Scan Station

| Feature | Benefit |
|---------|---------|
| TWAIN support | Works with most scanners |
| Batch scanning | Multiple documents at once |
| OCR integration | Make scans searchable |

### Import/Export Tool

| Feature | Benefit |
|---------|---------|
| Bulk operations | Move large amounts of data |
| Folder structure | Preserve organization |
| Metadata mapping | Maintain properties |

### E-mail Scanner

| Feature | Benefit |
|---------|---------|
| POP3 monitoring | Watch email accounts |
| Auto-routing | Send to specific folders |
| Attachment handling | Extract and store files |

---

## Installation Location

All module installers are located in:

```
[infoRouter installation directory]\Program Setups\
```
