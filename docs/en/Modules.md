# Modules and Add-ins / Introduction

infoRouter offers several modules and add-ins to assist in document capturing and processing.

These tools can be found in the "program setups" under the infoRouter application directory.

Addional Modules:

* Office Add-in
* Hot Folders
* Scan Station
* Import Export Tool
* E-mail Scanner
