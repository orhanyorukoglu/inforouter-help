# Creating Documents

There are several ways to create documents in infoRouter, from uploading existing files to creating content directly within the system.

---

## Creation Methods

| Method | Description | Best For |
|--------|-------------|----------|
| :material-upload: **[Upload](UploadingDocuments.md)** | Upload files from your computer | Existing documents |
| :material-file-document-edit: **[Form Templates](UsingFormTemplates.md)** | Create from predefined templates | Standardized forms |
| :material-language-html5: **HTML Editor** | Create using built-in editor | Simple web content |
| :material-microsoft-office: **[Office Add-in](OfficeAddin.md)** | Save directly from Office apps | Office documents |
| :material-folder-network: **[WebDAV](WebdavIntro.md)** | Save via mapped drive | Any file type |

---

## Uploading Documents

The most common way to add documents:

1. Navigate to the destination folder
2. Click **New** > **Upload**
3. Select files from your computer
4. Set properties (optional)
5. Click **Upload**

:material-arrow-right: [Uploading Documents](UploadingDocuments.md)

---

## Using Form Templates

Create standardized documents from templates:

1. Navigate to the destination folder
2. Click **New** > **Document from Template**
3. Select a template
4. Fill in the form fields
5. Click **Save**

:material-arrow-right: [Using Form Templates](UsingFormTemplates.md)

---

## Creating HTML Documents

Use the built-in editor for simple content:

1. Navigate to the destination folder
2. Click **New** > **HTML Document**
3. Use the editor to create content
4. Click **Save**
