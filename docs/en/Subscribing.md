# Subscribing to Documents and Folders

Subscribe to documents and folders to receive automatic notifications when changes occur. This is essential for staying informed about critical documents like contracts, proposals, or project deliverables.

---

## Why Subscribe?

!!! tip "Stay Informed"
    Subscriptions automatically notify you when:

    - Documents are revised or updated
    - New documents are created
    - Documents are approved or rejected
    - Other significant events occur

---

## Subscribing to Documents

1. Select the document(s) by clicking the checkboxes next to them
2. Go to **Edit** → **Subscribe**
3. The Subscribe window appears
4. Check the events you want to be notified about
5. Click **OK**

---

## Subscribing to Folders

1. Navigate to the folder
2. Go to **Edit** → **Subscribe**
3. The Subscribe window appears
4. Check the events you want to be notified about
5. Click **OK**

!!! note "Folder Subscriptions"
    When you subscribe to a folder, you'll be notified about changes to any document within that folder.

---

## Available Events

You can subscribe to be notified of these events:

| Event | Description |
|-------|-------------|
| **New Documents** | Documents created in the folder |
| **Read** | Documents viewed |
| **Change** | New versions created |
| **Update** | Property changes |
| **Check Out** | Documents checked out |
| **Check In** | Documents checked in |
| **Approve** | Documents approved |
| **Reject** | Documents rejected |
| **Comment** | Comments added |
| **Move** | Documents moved |
| **Delete** | Documents deleted |
