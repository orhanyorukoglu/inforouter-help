# Additional Modules / infoRouter Add-in for MS Office

infoRouter Add-in for MS Office

The infoRouter Add-in for MS Office allows users to interact with infoRouter from within MS Office products.

Supported MS Office Products:

* MS Word
* MS Outlook
* MS Excel
* MS PowerPoint

Using the add-in, you can open, check out, check in infoRouter documents directly from within the above MS Office products.
