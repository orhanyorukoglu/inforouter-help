# MS Office Add-in

The infoRouter Add-in for MS Office allows users to interact with infoRouter directly from within Microsoft Office applications.

---

## Supported Applications

| Application | Description |
|-------------|-------------|
| :material-microsoft-word: **MS Word** | Create and edit Word documents |
| :material-microsoft-outlook: **MS Outlook** | Register emails to infoRouter |
| :material-microsoft-excel: **MS Excel** | Manage Excel spreadsheets |
| :material-microsoft-powerpoint: **MS PowerPoint** | Work with presentations |

---

## Key Features

Using the add-in, you can:

- :material-file-download: **Open** documents directly from infoRouter
- :material-lock-open: **Check Out** documents for editing
- :material-lock: **Check In** documents with new versions
- :material-email: **Register emails** from Outlook to infoRouter
- :material-upload: **Save** new documents directly to infoRouter

!!! tip "Seamless Integration"
    The add-in adds infoRouter functionality directly to the Office ribbon, making document management seamless within your normal workflow.
