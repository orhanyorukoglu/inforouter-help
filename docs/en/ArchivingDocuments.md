# Archiving Documents and Folders

Archive individual documents and folders to remove them from the active view while retaining full access when needed.

---

## How to Archive a Document

1. Navigate to the document
2. Click the **Action** icon
3. Select **Archive**
4. Confirm the action

---

## How to Archive a Folder

1. Navigate to the folder
2. Click the **Action** icon on the folder
3. Select **Archive**
4. Choose whether to archive subfolders and documents
5. Confirm the action

---

## Effects of Archiving

| Aspect | Behavior |
|--------|----------|
| :material-eye-off: **Visibility** | Archived items are hidden from default view |
| :material-magnify: **Search** | Not included in search unless specified |
| :material-lock-open: **Access** | Still accessible when archives are visible |
| :material-shield: **Security** | All permissions preserved |

---

## Viewing Archived Items

To see archived content:

1. Go to **View** menu
2. Select **Library View Options**
3. Enable **Show Archive Libraries**

!!! tip "Visual Indicator"
    Archived items appear in **purple** when visible.

---

## Unarchiving Items

1. Make archives visible (see above)
2. Navigate to the archived item
3. Click the **Action** icon
4. Select **Unarchive**
