# Additional Modules / infoRouter Import Export Tool

infoRouter Import Export tool can be installed on any workstation to import documents into infoRouter. You can also export documents out of infoRouter using the same tool.

The infoRouter Import Export tool can be downloaded directly from the infoRouter Tools / Client Add-ins and Programs menu

For more information on this tool, please refer to these guides:

[Import Export tool installation Guide](https://www.infoRouter.com/downloads/V80/infoRouter_Import_Export_Tool_Installation_Guide.pdf)

[Import Export User's Guide](https://www.infoRouter.com/downloads/V80/infoRouter_Import_Export_Tool_Users_Guide.pdf)
