# Import/Export Tool

The infoRouter Import/Export Tool is a desktop application for bulk importing documents into infoRouter or exporting documents out of the system.

---

## Overview

This tool provides a graphical interface for transferring large numbers of documents between your local file system and infoRouter. It's ideal for:

- Initial document migration
- Bulk uploads
- Document backup/export
- Offline document access

---

## Key Features

| Feature | Description |
|---------|-------------|
| :material-upload: **Bulk Import** | Upload many documents at once |
| :material-download: **Bulk Export** | Download documents from infoRouter |
| :material-folder-multiple: **Folder Structure** | Preserve folder hierarchies during transfer |
| :material-form-textbox: **Metadata** | Apply properties during import |
| :material-api: **Web Services API** | Uses infoRouter's API for fast, reliable transfers |

---

## Getting the Software

The Import/Export Tool can be downloaded from within infoRouter:

1. Navigate to **Tools** menu
2. Select **Client Add-ins and Programs**
3. Download Import/Export Tool
