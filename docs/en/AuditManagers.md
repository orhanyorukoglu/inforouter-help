# Administrative Functions and Roles / Audit Managers

"Audit Managers" is a system user group that comes built-in to infoRouter. This system user group cannot be deleted. Its members are assigned by the system administrator and these members inherit special permissions that allow them to perform certain functions in infoRouter without restrictions.

Audit Managers can perform the following functions:

1. Navigate to the "Control Panel" and access system level Audit Logs and Reports
2. Navigate to any library and access folder and document level Audit Logs and Reports.
