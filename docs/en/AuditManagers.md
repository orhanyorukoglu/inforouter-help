# Audit Managers

**Audit Managers** is a built-in system user group in infoRouter. Members of this group have special permissions to access audit logs and reports throughout the system.

!!! note "System Group"
    This system user group cannot be deleted. Members are assigned by the System Administrator.

---

## Audit Manager Capabilities

Members of the Audit Managers group can:

| Function | Description |
|----------|-------------|
| :material-file-search: **System-level Audit Logs** | Access logs from the Control Panel |
| :material-folder-search: **Folder-level Audit Logs** | View audit trails for any folder |
| :material-file-document-edit: **Document-level Audit Logs** | View audit trails for any document |
| :material-chart-bar: **Reports** | Generate and view audit reports |

---

## Scope of Access

| Area | Access Level |
|------|--------------|
| **Control Panel Logs** | Full read access |
| **All Libraries** | Audit log access |
| **All Folders** | Audit log access |
| **All Documents** | Audit log access |

---

## Accessing Audit Logs

### System-Level Logs

1. Navigate to **Control Panel**
2. Access **Audit Logs and Reports**

### Library/Folder/Document Logs

1. Navigate to any library
2. Access **Logs & Reports** from folder or document properties

---

## Becoming an Audit Manager

The System Administrator assigns users to the Audit Managers group:

1. Go to **Control Panel**
2. Navigate to **System User Groups**
3. Select **Audit Managers**
4. Add members to the group
