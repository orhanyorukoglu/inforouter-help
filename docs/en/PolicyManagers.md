# Administrative Functions and Roles / Policy Managers

"Policy Managers" is a system user group that comes built-in to infoRouter. This system user group cannot be deleted. Its members are assigned by the system administrator and these members inherit special permissions that allow them to perform certain functions in infoRouter without restrictions.

Policy Managers can perform the following functions:

1. Edit Library policies for all Libraries.
2. Edit Authentication and Password Policy Settings
