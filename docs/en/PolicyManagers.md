# Policy Managers

**Policy Managers** is a built-in system user group in infoRouter. Members of this group have special permissions to manage policies across all libraries.

!!! note "System Group"
    This system user group cannot be deleted. Members are assigned by the System Administrator.

---

## Policy Manager Capabilities

Members of the Policy Managers group can:

| Function | Description |
|----------|-------------|
| :material-library-shelves: **Edit Library Policies** | Modify policies for all libraries |
| :material-shield-key: **Edit Authentication Settings** | Configure authentication methods |
| :material-lock: **Edit Password Policies** | Set password requirements and expiration rules |

---

## Scope of Access

| Area | Access Level |
|------|--------------|
| **All Libraries** | Full policy editing access |
| **Authentication Settings** | Full configuration access |
| **Password Policies** | Full configuration access |

---

## Becoming a Policy Manager

The System Administrator assigns users to the Policy Managers group:

1. Go to **Control Panel**
2. Navigate to **System User Groups**
3. Select **Policy Managers**
4. Add members to the group
