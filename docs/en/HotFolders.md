# Hot Folders

infoRouter Hot Folders enables automatic document ingestion from designated file system folders into infoRouter.

---

## How It Works

The Hot Folders tool runs as a Windows service and continuously monitors specified folders. When new documents appear, they are automatically imported into designated infoRouter folders.

```mermaid
graph LR
    A[File System Folder] --> B[Hot Folders Service]
    B --> C[infoRouter Folder]
```

!!! tip "TIFF Conversion"
    If the "Convert TIFF to PDF" option is enabled in application settings, TIFF files are automatically converted to PDF during import.

---

## Common Use Cases

| Scenario | Description |
|----------|-------------|
| :material-fax: **Incoming Fax** | Automatically capture faxes as they arrive |
| :material-upload-network: **FTP Folders** | Import files uploaded via FTP |
| :material-scanner: **Scanned Documents** | Ingest output from network scanners |
| :material-cog: **Automated Jobs** | Capture documents generated by automated processes |

---

## Key Features

- :material-eye: **Continuous Monitoring** — Watches folders 24/7
- :material-folder-multiple: **Multiple Folders** — Monitor several folders simultaneously
- :material-server: **Service-Based** — Runs as a Windows service on any network machine
- :material-file-swap: **Auto-Conversion** — Optional TIFF to PDF conversion
