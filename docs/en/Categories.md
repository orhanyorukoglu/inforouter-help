# Categories / Saved Search

Categories are pre-defined search queries. Categories are used to group related documents under a common heading. Categories appear under the "Categories" Tab from the main Portal page.

To search for documents using Categories, simply perform the following:

1. Navigate to the "Categories" tab.
2. Look for a Category name that best describes the type of document you are searching for.
3. Click on the category header to display a list of documents that fall under the described category
