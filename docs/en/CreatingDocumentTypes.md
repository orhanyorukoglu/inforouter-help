# Creating Document Types

Define document types that match your organization's classification needs.

---

## What Are Document Types?

Document types categorize documents by their content or purpose. Examples include:

| Document Type | Description |
|---------------|-------------|
| Invoice | Financial billing documents |
| Contract | Legal agreements |
| Policy | Organizational policies and procedures |
| Report | Business or technical reports |
| Proposal | Project or sales proposals |

---

## How to Create a Document Type

1. Navigate to the **Control Panel**
2. Click **Document Types**
3. Click **New** in the menu
4. Enter the **Document Type name**
5. Optionally, select a **Custom Property Set** from the dropdown
6. Click **OK**

---

## Document Type Settings

| Setting | Description |
|---------|-------------|
| **Name** | The display name for the document type |
| **Custom Property Set** | Links metadata requirements to this document type |

!!! info "Enforcing Metadata"
    When you link a Custom Property Set to a document type, users must enter the required metadata before they can assign that document type to a document.

---

## Using Document Types

Document types are used throughout infoRouter:

| Feature | How Document Types Are Used |
|---------|----------------------------|
| :material-upload: **Uploading** | Assign type when adding documents |
| :material-magnify: **Searching** | Filter searches by document type |
| :material-tag: **Organizing** | Group similar documents |
| :material-file-document: **Properties** | View or change document type |

---

## Editing Document Types

1. Navigate to the **Control Panel**
2. Click **Document Types**
3. Select the document type to edit
4. Modify settings as needed
5. Click **OK**

---

## Deleting Document Types

!!! warning "Deletion Impact"
    Before deleting a document type, consider that existing documents using this type will no longer have a valid type assigned.
