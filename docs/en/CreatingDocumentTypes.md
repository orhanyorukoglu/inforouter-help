# Administrative Functions / Creating Document Types

Each organization can define document types that best suit their environment. Document types can refer to just about anything. It may explain the individual document content or the department in which it was created. For more information on document types, visit [Document Types](DocumentTypes.md).

To Create or Edit Document types in infoRouter.

1. Navigate to the infoRouter Control Panel
2. Click on the Document Types link.
3. Click on the New menu.
4. In the window that appears, enter the Document Type name.
5. Select from the Custom Property drop down to enforce users to enter custom property (Meta) data before they can mark a document with the specified document type name.
