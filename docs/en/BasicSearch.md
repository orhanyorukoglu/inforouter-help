# Basic Search

Quickly find documents using simple keyword searches in infoRouter.

---

## Using Basic Search

Basic search is the fastest way to find documents when you know what you're looking for.

1. Enter your search term(s) in the **Search** box
2. Press **Enter** or click the **Search** button
3. Results are displayed in a list

---

## Search Behavior

| Behavior | Description |
|----------|-------------|
| :material-text-search: **Full-text search** | Searches within document content |
| :material-file-document: **Metadata search** | Searches document names and properties |
| :material-library: **Library scope** | Searches within your current library context |

---

## Tips for Better Results

!!! tip "Search Tips"
    - Use specific keywords rather than common words
    - Put phrases in quotes to search for exact matches: `"annual report"`
    - Use multiple keywords to narrow results

---

## Search Results

Results display:

| Column | Description |
|--------|-------------|
| **Name** | Document name (click to view) |
| **Type** | Document type/format |
| **Modified** | Last modification date |
| **Location** | Folder path |

---

## When to Use Advanced Search

Consider using [Advanced Search](AdvancedSearch.md) when you need to:

- Search by specific date ranges
- Filter by document properties
- Search within specific folders
- Use complex search operators
