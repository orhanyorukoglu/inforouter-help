# Deleting Categories

Remove categories that are no longer needed from the system.

---

## Who Can Delete Categories?

| Role | Can Delete |
|------|------------|
| System Administrator | :material-check: Yes |
| Administrators group | :material-check: Yes |
| Category & Search Administrators | :material-check: Yes |
| Regular Users | :material-close: No |

!!! note "Delete Icon Visibility"
    The delete icon only appears if you are a designated Search and Category Administrator.

---

## How to Delete a Category

1. Click the **Categories** tab
2. Locate the category you want to delete
3. Click the **Delete** icon (üóëÔ∏è) next to the category name
4. Confirm the deletion

---

## Considerations Before Deleting

!!! warning "Cannot Be Undone"
    Category deletion is permanent. Consider the following before deleting:

| Consideration | Details |
|---------------|---------|
| **User reliance** | Users may depend on this category for regular searches |
| **Documentation** | Update any documentation that references this category |
| **Alternatives** | Ensure users have alternative ways to find these documents |

---

## What Happens When Deleted

- The category is removed from the Categories tab
- The underlying search criteria is deleted
- Documents previously found by this category are **not affected**
- Users can no longer use this category to search
